# JDBC vs Spring JDBC

-------

### JDBC
- 많은 SQL 쿼리를 작성해야한다.
- 그리고 많은 Java 코드를 작성해야한다.

### Spring JDBC

- 많은 SQL 쿼리를 작성해야한다.
- JDBC 보단 적은 java 코드를 작성한다.


# JPA vs Spring Data JPA

### JPA 

- 쿼리를 작성하지 않아도된다.

### Spring 

- JPA 보다도 더 간단하다.


# Hibernate vs JPA


-----

### JPA

- 객체를 테이블로 매핑하는 방식을 정의
